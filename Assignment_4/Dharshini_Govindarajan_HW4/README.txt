## Task 1: Generating the tables for Lucene:

#### To Execute
1) Navigate to the folder, containing the file (Task1.java)
	Eg -> C:\Users\Desktop\Semester4\IR_HW4\IR_HW4\src\HW4
2) To compile and execute in default mode type the following (In Java/Eclipse terminal/cmd)
	javac Filename.java
	java Filename
3) Enter the location you want the Index to be generated
4) Enter the location of your corpus
5) Files are generated in the Folder called "Task1_Files"
