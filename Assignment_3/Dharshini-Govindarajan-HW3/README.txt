## Task 1: Generating the corpus:

#### To Execute
1) Navigate to the folder, containing the file (Task1.py)
	Eg -> C:\Documents\IR_WebCrawler\crawler\HW3
2) To execute in default mode type the following (In pythonScript/pycharm terminal/cmd)
	python Task1.py
3) To execute in Case Folding mode type the following (In pythonScript/pycharm terminal/cmd)
	python Task1.py "CF"
4) To execute in Remove Punctuation mode type the following (In pythonScript/pycharm terminal/cmd)
	python Task1.py "RP"
5) Files are generated in the Folder called "Corpus" (IR_WebCrawler\HW3\Corpus)
